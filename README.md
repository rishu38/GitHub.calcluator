# Calculator in C

This is a simple calculator program written in C. It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## 🚀 Features
- User input based calculator
- Supports +, -, *, / operations
- Uses switch case
- Handles division by zero error

## 🛠️ Technologies Used
- C Programming Language

## ▶️ How to Run
1. Open the program in CodeBlocks / VS Code
2. Compile the code
3. Run the program
4. Enter numbers and operator

## 💡 Example
Enter first number: 10  
Enter operator: +  
Enter second number: 5  
Output: 15
